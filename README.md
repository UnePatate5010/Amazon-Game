# Game of the Amazons

Game of the Amazons is a board games similar to chess. Check [Wikipedia](https://en.wikipedia.org/wiki/Game_of_the_Amazons) for rules (and more).

This is a school project. The objective was to implement the Games of the Amazons with a player/server systems using dynamic libraries. All students groups (~25 groups) were ranked on a ladder depending on victories and defeats against other groups' players.
We ranked 4th with a [minimax algorithm](https://en.wikipedia.org/wiki/Minimax) and a well cooked heuristic evaluation function.
